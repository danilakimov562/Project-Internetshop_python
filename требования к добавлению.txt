Для добавления клиента:
    ID клиента должен быть уникальным целым числом (не должен уже существовать в базе).
    Имя клиента — непустая строка.
    Email должен быть корректным с точки зрения формата (например, содержать символ "@", доменное имя и т.п.). Проверяется регулярным выражением.
    Телефон должен соответствовать формату номера телефона, например, начинаться с "+" и содержать от 10 до 15 цифр (в соответствии с регулярным выражением).
    Все поля должны быть заполнены (не пустые).
Для добавления товара:
    ID товара должен быть уникальным целым числом (не должен уже существовать в базе).
    Название товара — непустая строка.
    Цена товара — число с плавающей точкой, неотрицательное (больше или равно нулю).
    Все поля должны быть заполнены корректно.
Для добавления заказа:
    ID заказа должен быть уникальным целым числом (не должен уже существовать в базе).
    ID клиента, к которому относится заказ, должен существовать в базе клиентов.
    Список ID товаров должен содержать по крайней мере один товар; все указанные ID товаров должны существовать в базе товаров.
    Дата заказа вводиться в ручную и должна быть в формате ДД-ММ-ГГГГ или наоборот.
